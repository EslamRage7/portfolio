# Fedaa Amin Portfolio (Vite + React)

## Scripts

- `npm run dev`: start local development server
- `npm run build`: create production build in `dist/`
- `npm run preview`: preview production build locally

## Notes

- The project was migrated from Create React App to Vite for faster startup and builds.
- All project assets (logo, images, videos, PDF) remain unchanged and continue to work from `src/Components`.
